## LangChain: Deep Agents Deploy (beta)
LangChain lanzó "Deep Agents Deploy" en beta como una herramienta para desplegar harnesses de agentes abiertos y agnósticos al proveedor en entornos de producción. La solución agrupa la orquestación del agente, endpoints para memoria, MCP/A2A y un servidor de despliegue escalable, permitiendo a los equipos desplegar agentes con control total sobre memoria y herramientas. Está diseñada para integrarse con múltiples proveedores de modelo y sandboxes (LangSmith, Modal, Runloop, etc.), y ofrece un flujo CLI simplificado para pasar de desarrollo a despliegue. [Fuente: blog de LangChain](https://blog.langchain.com/deep-agents-deploy-an-open-alternative-to-claude-managed-agents/). ([blog.langchain.com](https://blog.langchain.com/deep-agents-deploy-an-open-alternative-to-claude-managed-agents/?utm_source=openai))

## AWS: Agent Registry (AgentCore) en preview
AWS anunció en preview el "Agent Registry" dentro de Amazon Bedrock AgentCore, un catálogo gestionado para indexar, descubrir y gobernar agentes, herramientas, MCP servers y skills en la organización. El servicio ofrece búsqueda semántica y por palabras clave, workflows de aprobación, auditoría con CloudTrail y acceso MCP para invocar recursos desde IDEs o agentes; la idea es frenar la "sprawl" de agentes y facilitar el reuso entre equipos. Está disponible en regiones donde AgentCore está habilitado y se puede usar vía consola, API o CLI. [Fuente: anuncio oficial de AWS](https://aws.amazon.com/about-aws/whats-new/2026/04/aws-agent-registry-in-agentcore-preview/). ([aws.amazon.com](https://aws.amazon.com/about-aws/whats-new/2026/04/aws-agent-registry-in-agentcore-preview/?utm_source=openai))

## OpenClaw 2026.4.9: nueva versión con 'Dreaming' y endurecimiento de seguridad
El proyecto OpenClaw publicó la versión v2026.4.9, que introduce mejoras en el sistema de memoria ("Dreaming") con backfill REM histórico y una interfaz de diario/timeline para gestionar recuerdos, además de varias correcciones y endurecimientos de seguridad (SSRF, sanitización de ejecuciones remotas, packaging y parches en integraciones). La release también mejora QA/evaluación de personajes y arregla múltiples problemas de estabilidad en gateways y proveedores. Ideal para quienes siguen frameworks de agentes open-source y necesitan memoria duradera y control de seguridad más fino. [Fuente: release tag / resumen de cambios en GitHub].(https://github.com/openclaw/openclaw/releases/tag/v2026.4.9). ([newreleases.io](https://newreleases.io/project/github/openclaw/openclaw/release/v2026.4.9?utm_source=openai))

## Microsoft: actualización de abril 2026 en Power Platform (integración de agentes y herramientas AI)
Microsoft publicó la nota "What’s new in Power Platform: April 2026" donde recoge múltiples novedades: integración de datos y experiencias de Power Apps con Microsoft 365 Copilot, soporte en preview para construir canvas apps con agentes de codificación (MCP Authoring Plugin), y disponibilidad global para páginas generativas con soporte de herramientas externas. También detalla mejoras de gobernanza, alertas y métricas para apps y nuevas capacidades en Copilot Studio y administración de agentes. Es una tanda de cambios pensada para acercar el desarrollo low‑code a flujos asistidos por agentes y APIs de IA. Perplexity anunció el concurso "Billion Dollar Build", un programa de ocho semanas en el que equipos usarán Perplexity Computer para construir startups con potencial de llegar a $1B. La iniciativa ofrece hasta $1M en inversión por ganador y hasta $1M en créditos de compute; la inscripción arranca el 14 de abril y el evento final de pitch está fijado para junio.
