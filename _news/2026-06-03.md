## Microsoft Build 2026: apertura y enfoque en agentes
Microsoft abrió Build 2026 con un mensaje claro: los agentes AI pasan a ser la prioridad central para plataformas de desarrollo y herramientas de productividad.  
La keynote destacó nuevas APIs, integraciones de gobernanza y especificaciones abiertas para controlar el comportamiento de agentes en producción.  
El anuncio subraya el empeño de Microsoft en ofrecer un stack integrado (modelos, runtime, observabilidad y control) para construir aplicaciones agentic.  
Para desarrolladores esto implica nuevas herramientas y rutas de integración con Copilot, Foundry y Fabric dentro del ecosistema Microsoft.  
[Fuente: Microsoft Blog — Microsoft Build 2026: Be yourself at work](https://blogs.microsoft.com/blog/2026/06/02/microsoft-build-2026-be-yourself-at-work/). ([blogs.microsoft.com](https://blogs.microsoft.com/blog/2026/06/02/microsoft-build-2026-be-yourself-at-work/?utm_source=openai))

## Microsoft Foundry: más piezas para producir agentes
Microsoft Foundry presentó en Build mejoras en runtime, memoria, grounding, observabilidad y herramientas de gobernanza dirigidas a agentes en producción.  
Las novedades buscan cerrar la brecha entre prototipos de agentes y despliegues robustos, con énfasis en seguimiento, pruebas y controles de políticas.  
Foundry avanza hacia un entorno integrado donde modelos, datos y runtime se gestionan con trazabilidad para auditoría y seguridad.  
Esto facilita la transición de prototipos a servicios gestionados dentro del ecosistema Microsoft.  
[Fuente: Microsoft Foundry Blog — What's new in Microsoft Foundry | Build Edition](https://devblogs.microsoft.com/foundry/whats-new-in-microsoft-foundry-build-2026/). ([devblogs.microsoft.com](https://devblogs.microsoft.com/foundry/whats-new-in-microsoft-foundry-build-2026/?utm_source=openai))

## Windows: herramientas para agentes y AI local
Microsoft detalló en el Windows Developer Blog novedades para convertir Windows en una plataforma que soporte agentes locales y experiencias AI avanzadas.  
Se anunciaron runtimes y APIs diseñadas para ejecutar agentes con mejores controles de seguridad y capacidades multimodales en el dispositivo.  
Microsoft enfatiza experiencias "local-first" que combinan confianza, rendimiento y herramientas para depuración y gobernanza en Windows.  
La apuesta busca posicionar Windows como un entorno de referencia para agentes que requieren integración con hardware acelerado.  
[Fuente: Windows Developer Blog — Build 2026: Furthering Windows as the trusted platform for development](https://blogs.windows.com/windowsdeveloper/2026/06/02/build-2026-furthering-windows-as-the-trusted-platform-for-development/). ## Visual Studio: próximos cambios y prioridades para desarrolladores
El equipo de Visual Studio publicó su hoja de ruta para Build 2026 con mejoras enfocadas en productividad con modelos, integración de Copilot y soporte del stack .NET.

## OpenAI: modelos Frontier y Codex disponibles en AWS
OpenAI anunció que sus modelos frontier y su servicio Codex están ahora disponibles a través de Amazon Bedrock en AWS, ampliando opciones de despliegue.  
La disponibilidad en AWS permite a clientes usar los modelos de OpenAI dentro del ecosistema de servicios y controles de AWS.  
La noticia refuerza la opción multicloud para equipos que buscan integrar modelos avanzados en infraestructuras gestionadas por AWS.  
[Fuente: OpenAI — OpenAI frontier models and Codex are now available on AWS](https://openai.com/index/openai-frontier-models-and-codex-are-now-available-on-aws/). ([openai.com](https://openai.com/index/openai-frontier-models-and-codex-are-now-available-on-aws/?utm_source=openai))

## NVIDIA presenta RTX Spark Superchip en Computex
NVIDIA reveló en Computex el RTX Spark, un "superchip" pensado para PCs y laptops con foco en cargas AI locales y soporte para experiencias agentic en el dispositivo.  
Las coberturas técnicas describen combinaciones de CPU ARM, GPU Blackwell y memoria unificada para cargas de modelos grandes y contextos extensos.  
La propuesta impulsa la tendencia hacia PCs capaces de ejecutar tareas de razonamiento y agentes con menor dependencia de la nube.

## AMD amplía la disponibilidad de la Radeon RX 9070 GRE
AMD anunció (vía coberturas de Computex) que la Radeon RX 9070 GRE, antes limitada a ciertos mercados, estará disponible globalmente con precio orientativo.  
La expansión hace más accesible una opción de GPU para homelabs y estaciones de trabajo orientadas a gráficos y cargas de IA de rango medio.  
Para quienes montan NAS/mini-PCs o rigs locales esto significa más alternativas de coste-rendimiento fuera del segmento tope de gama.

## Akamai y NVIDIA: seguridad integrada para fábricas de IA
Akamai y NVIDIA anunciaron una colaboración para añadir controles de seguridad a la infraestructura de "AI factories", con enfoque en segmentación y monitoreo a nivel infra.  
La iniciativa pretende detectar y contener comportamientos anómalos de agentes y modelos sin penalizar significativamente el rendimiento de cómputo.  
Este tipo de integraciones es relevante para centros que ejecutan pipelines agentic a gran escala donde la contención es crítica.  
[Fuente: Taiwan News (resumen de nota de prensa) — Akamai Brings Security Inside AI Factories with NVIDIA](https://www.taiwannews.com.tw/news/6375322). ([taiwannews.com.tw](https://www.taiwannews.com.tw/news/6375322?utm_source=openai))

## Cisco presenta "Cloud Control", plataforma agentic para operaciones y defensa
Cisco presentó en Cisco Live la plataforma Cisco Cloud Control, orientada a operar y defender infraestructuras críticas con agentes integrados y controles humanos.  
La solución agrupa redes, seguridad, observabilidad y automatización bajo una visión "AgenticOps" para gestionar ataques y operaciones a velocidad de máquina.  
Cisco anuncia además capacidades de protección en tiempo de ejecución pensadas para reducir ventanas de exposición y automatizar respuestas.  
[Fuente: Cisco Newsroom — Cisco Unveils Agentic Platform for Operating and Defending Critical IT Infrastructure](https://newsroom.cisco.com/c/r/newsroom/en/us/a/y2026/m06/cisco-unveils-agentic-platform-for-operating-and-defending-critical-it-infrastructure.html). ## Google rompe tierra para nuevo centro de datos en Horndal (Suecia)
Google anunció la construcción de un nuevo centro de datos en Horndal, Suecia, con diseño orientado a recuperación de calor y sostenibilidad.  
El proyecto incluye inversión en la comunidad local y se alinea con la demanda creciente de capacidad para servicios de Search, Cloud, Workspace y YouTube.  
Desde la perspectiva cloud, la expansión fortalece la infraestructura regional para cargas de datos y aplicaciones AI/ML.

## Workday y Google: integración de agentes en Gemini Enterprise
Workday anunció integración de capacidades de HR y finanzas dentro de Gemini Enterprise para habilitar flujos de trabajo agentic empresariales.  
La integración apunta a facilitar flujos A2A (agent-to-agent) y A2UI (agent-to-UI), conectando procesos críticos con modelos y controles de empresa.  
Para organizaciones, esto supone un paso hacia automatizaciones más profundas en procesos de negocio con modelos gestionados en la nube.
