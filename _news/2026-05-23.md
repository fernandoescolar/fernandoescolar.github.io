## Azure anuncia preview público de red entre clusters para Fleet Manager (Cilium)
Microsoft publica el anuncio de la vista previa pública de una solución de networking "cross‑cluster" para Azure Kubernetes Fleet Manager basada en Cilium. La característica permite tráfico east‑west entre pods de clusters distintos como si fueran locales, facilita discovery global de servicios mediante una simple anotación y centraliza telemetría y políticas a nivel de flota. Microsoft detalla requisitos (Azure CNI powered by Cilium, Advanced Container Networking Services) y pasos de activación; es una apuesta clara por simplificar arquitecturas multi‑cluster gestionadas en AKS. ([azure.microsoft.com](https://azure.microsoft.com/en-us/blog/powering-multi-cluster-workloads-with-seamless-cross-cluster-networking-for-azure-kubernetes-fleet-manager/))  
[Fuente: Microsoft Azure Blog — Cross‑cluster networking para Azure Kubernetes Fleet Manager](https://azure.microsoft.com/en-us/blog/powering-multi-cluster-workloads-with-seamless-cross-cluster-networking-for-azure-kubernetes-fleet-manager/)

## Envoy Gateway lanza v1.0
El proyecto Envoy Gateway publica la versión `v1.0` de su gateway, marcando un hito de estabilidad y conjunto de características para quienes construyen gateways API basados en la pila Envoy. La release incluye mejoras en la integración con Kubernetes, seguridad y personalización del chart Helm; el proyecto destaca compatibilidad con políticas de seguridad y mayor flexibilidad para definiciones de `podSecurityContext` en el Helm chart. Esta versión es relevante para equipos cloud‑native que buscan gateways ligeros y alineados con el ecosistema Envoy/EnvoyProxy. ([gateway.envoyproxy.io](https://gateway.envoyproxy.io/news/releases/v1.0/))  
[Fuente: Envoy Gateway — Announcing Envoy Gateway v1.0](https://gateway.envoyproxy.io/news/releases/v1.0/)

## GIGABYTE recibe premios COMPUTEX 2026 por hardware orientado a IA
En COMPUTEX 2026 GIGABYTE ha sido reconocido con Best Choice Awards por varias piezas de hardware orientadas a cargas locales de IA (placas base, eGPU y portátiles diseñados para sostenimiento de inferencia/entrenamiento ligero). Los productos premiados subrayan la dirección del mercado hacia mini‑PCs y placas madre explícitamente optimizadas para workloads de IA en entornos de escritorio y homelab, con atención a refrigeración, I/O y soporte para aceleradores. Esto es relevante para quien planifique un homelab con capacidad de inferencia local. ([engineering.com](https://www.engineering.com/gigabyte-wins-computex-2026-awards-for-ai-hardware/))
