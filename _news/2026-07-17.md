## Microsoft: Patch Tuesday de julio 2026 (gran oleada de parches)
Microsoft publicó el paquete de seguridad de julio 2026 correspondiente al Patch Tuesday, con un volumen sin precedentes de correcciones. Los análisis de la industria informan que la entrega incluye cientos de arreglos (informes hablan de ~622 CVEs según el recuento del Security Update Guide), entre ellos varias vulnerabilidades de explotación activa y varias de criticidad elevada que requieren priorización en entornos expuestos. Las páginas de soporte de Microsoft y las notas acumulativas de julio recogen los KB específicos publicados para distintas versiones de Windows y productos asociados; como siempre, los administradores deben priorizar parches según exposición e impactar primero sistemas con servicios públicos o con evidencia de explotación. [Fuente: Microsoft Support (KBs de julio 14, 2026) / Análisis de seguridad]. 

## .NET: .NET 11 Preview 6 y actualizaciones de servicio de julio 2026
El equipo de .NET publicó noticias de julio que incluyen la disponibilidad de .NET 11 Preview 6 y un resumen de las actualizaciones de servicio de julio (servicing updates) para .NET y .NET Framework. Preview 6 enumera mejoras en runtime, SDK, librerías y en ASP.NET Core; por separado, las notas de servicing describen correcciones de seguridad y de calidad para las ramas soportadas (incluyendo versiones de ASP.NET Core y EF Core). Se recomienda a desarrolladores y equipos de CI probar Preview 6 en entornos de prueba y planificar la aplicación de los servicing updates en producción según la política de actualizaciones. [Fuente: .NET Blog]. 

## Microsoft Foundry / Agent Framework: SDKs y hosting para agentes (C# y Python)
Microsoft Foundry ha continuado empujando integraciones para agentes: la documentación y guías públicas muestran que el Microsoft Agent Framework ahora tiene soporte y hosting integrado para Foundry, con guías de despliegue de agentes alojados y paquetes para .NET y Python. Las páginas oficiales describen cómo empaquetar agentes, usar clientes Foundry y ejecutar modelos localmente con Foundry Local; también hay ejemplos y quickstarts para desplegar agentes como contenedores gestionados por el servicio Foundry. Esto refuerza la pila de Microsoft para desarrollar y operar agentes multietapa usando C# o Python. [Fuente: Microsoft Learn (Foundry / Agent Framework)]. 

## Model Context Protocol (MCP): release candidate 2026-07-28 y cambios importantes
La especificación MCP tiene una release candidate pública para la revisión de la comunidad con fecha objetivo 2026-07-28. La revisión prepara un núcleo sin estado, un framework de extensiones, mejoras en Tasks, soporte formal para aplicaciones MCP y una política de deprecación más explícita; los proveedores y herramientas del ecosistema ya están publicando guías y análisis sobre el impacto de estos cambios. Los equipos que integran agentes, gateways MCP o pasarelas de gobernanza deberían revisar el RC y prepararse para adaptaciones de seguridad y de compatibilidad. [Fuente: blog.modelcontextprotocol / entradas de release].
