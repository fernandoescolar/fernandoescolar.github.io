## OpenAI lanza la familia GPT-5.6 (Sol, Terra, Luna) y la integra en ChatGPT/Codex/API
OpenAI ha movido la familia de modelos GPT-5.6 a disponibilidad pública, presentando tres variantes orientadas a distintos usos: un modelo tope de gama (Sol), uno intermedio para tareas de trabajo (Terra) y una opción más económica y rápida (Luna). La compañía destaca mejoras en razonamiento, seguridad y capacidades de codificación; la familia se expone dentro de ChatGPT (incluyendo ChatGPT Work), Codex y la API, y la documentación oficial y notas de producto describen el acceso y los escenarios de uso para equipos y desarrolladores. ([openai.com](https://openai.com/index/gpt-5-6/?utm_source=openai)) [Fuente](https://openai.com/index/gpt-5-6/)

## Anthropic: Claude Code Desktop incorpora un navegador integrado sandbox para interacción web
Anthropic actualizó Claude Code Desktop con un navegador integrado y sandboxeado que permite a los flujos de trabajo automatizados interactuar con páginas web en un entorno aislado (con opciones para persistencia de sesión). Las notas de lanzamiento y coberturas recientes explican que la funcionalidad apunta a mejorar las tareas de scraping controlado, pruebas de apps web y workflows de agentes manteniendo límites sobre credenciales y datos sensibles. ([support.claude.com](https://support.claude.com/en/articles/12138966-release-notes?debug=1&debug=true&debug_url=1&utm_source=openai)) [Fuente](https://support.claude.com/en/articles/12138966-release-notes)

## Google Cloud: novedades en Compute Engine — imágenes optimizadas y series C4N para cargas de red intensivas
Google Cloud publicó notas de producto que incluyen una vista previa de "Advanced Compute Images" —imágenes preparadas y optimizadas para IA/ML y HPC— y la disponibilidad general de la serie de máquinas `C4N` para Compute Engine y GKE, orientadas a cargas con alto uso de red y I/O. Las entradas de release notes describen los escenarios recomendados para despliegues de inferencia y sistemas distribuidos, además de mejoras recientes en herramientas de provisión y configuración. ([docs.cloud.google.com](https://docs.cloud.google.com/compute/docs/release-notes?authuser=14&utm_source=openai)) [Fuente](https://docs.cloud.google.com/compute/docs/release-notes)

## Hardware / homelab: aparición de variantes "Super" de la serie RTX 50 en calculadora de Seasonic
